# Game Hub
This is an interactive desktop application for storing a list of different click-and-play games. It was developed using the Python Tkinter library for front end user-interface, as well as the Pygame library for game functionality.

## Instructions
**1. Clone repository and enter directory:**  
  git clone https://github.com/aaronnewmanj/Game-Hub.git  
  cd Game-Hub   

**2. Run Program:**  
  python3 GameHub.py  
  or  
  python GameHub.py
